# BlackHole3DProject


Projeto de demonstração em Java + Java3D.
Arquivos principais em src/ e as classes do pacote 'simulation'.

- Exemplos de como compilar (linha de comando):
    javac -cp /path/to/j3dcore.jar:/path/to/vecmath.jar:/path/to/j3dutils.jar -d out src/*.java src/simulation/*.java
    java -cp out:/path/to/j3dcore.jar:/path/to/vecmath.jar:/path/to/j3dutils.jar Main
